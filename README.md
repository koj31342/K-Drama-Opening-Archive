# 한국 사극 드라마 오프닝 아카이브

한국 사극 드라마 오프닝, 티저 예고편 아카이브 프로젝트
링크 : https://koj31342.github.io/K-Drama-Opening-Archive/

## 기능

- 제목, 연도, 방송사, 시대, 고유 DB ID를 통한 검색 기능 제공
- 현재 개발자 개인의 오프닝 소유 여부 확인 가능
- TMDB API를 이용한 작품별 썸네일 제공
- 스프레드시트 DB CSV 파싱을 통한 실시간 연동

## 기타

- 본 프로젝트는 CHATGPT를 사용하여 만든 프로젝트입니다.
- 저작권 이슈로 오프닝 및 티저 예고편을 제공하지 않습니다.
- 비영리적으로 사용되며 정보 확인 목적의 프로젝트입니다.
- 오프닝 및 티저 예고편은 워터마크가 없는 준수한 화질의 영상만 소유하였습니다.
- 웹에 검색되는 워터마크가 있는 오프닝 자료는 개인적으로 소장 가치가 없다고 판단하여 소장하지 않았습니다.
